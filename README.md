# 先开始模型选型和部署吧
 骨骼与手部、面部关键点 ： MediaPipe Holistic  
  物品目标检测 ： YOLOv8-Nano/Small
   手部时序动作识别（区分「敲键盘」还是「握手柄」）。先用手部运动统计规则（指尖速度方差、运动频率）暂时区分“高频击键”和“稳定握持”，后续再迭代为模型
    注意力估计（在「看」哪个物品）：
    头部姿态：从面部关键点（至少鼻尖、眼角、耳朵）解算欧拉角（pitch, yaw, roll），得到视线方向向量。

视线与物品关联：

    计算视线向量与各物品检测框中心/三维估计位置的夹角。

    结合距离，为每帧计算注意力得分，得分最高且超过阈值的物品即为“当前注视物体”。

    可辅助判断“看手机”“看电脑屏幕”。
